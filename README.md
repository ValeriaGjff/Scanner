# Scanner

Проект для Android.  
Приложение использует **Google ML Kit** для распознавания штрих- и QR-кодов с камеры.  

## Возможности
- Сканирование QR- и штрих-кодов
- Отображение результата
- Простое и понятное использование

## Стек
- Android (Java)
- Google ML Kit
